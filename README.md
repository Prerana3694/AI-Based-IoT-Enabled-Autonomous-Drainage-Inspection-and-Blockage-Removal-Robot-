# AI-Based-IoT-Enabled-Autonomous-Drainage-Inspection-and-Blockage-Removal-Robot-
AI-based IoT autonomous drainage inspection and blockage removal robot. It uses sensors to detect blockage and overflow, enables real-time monitoring, and automates cleaning operations. Designed for smart drainage systems to improve efficiency and reduce manual effort.
# AI-Based Drainage Inspection Robot

## Description
This project detects blockage and overflow in drainage systems using sensors and AI.

## Features
- Blockage detection
- Real-time monitoring
- IoT-based alert system

## Technologies Used
- Arduino / Microcontroller
- Sensors
- AI concepts
